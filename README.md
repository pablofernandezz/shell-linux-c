# shell3-FINAL
***** INSTRUCCIONES DEL GIT ********


Para subir los archivos, seguid estos pasos.

Si es la primera vez que hacéis esto...

1. Cread una carpeta
2. Abrid esa carpeta con VSCODE
3. Abrid la terminal
4. Dadle a git init
5. Dadle a git clone (url). Esto creara una carpeta llamada C-Shell dentro de vuestra carpeta.
6. En la terminal, dadle a "cd C-Shell" (os meteis en la carpeta, comprobad esto)
7. Ahora podeis hacer cosas...
(PUSH)
1. Poned en el ReadMe los cambios que hagais y si teneis problemas
2. Una vez querais subir algo, id a la terminal de nuevo. COMPROBAD QUE ESTAIS EN LA CARPETA C-SHELL
3. git add .
4. git commit -m "funciones q tocasteis"
5. git push origin main
(PULL)
1. Os poneis en vuestra carpeta C-SHELL MUY IMPORTANTE
2. git pull
